# Twine Documentation

This repository contains the documentation for the Twine project, built using Rust's `mdBook` tool.

## Building the Documentation

To build the documentation locally, you'll need to have Rust and Cargo installed. Then, you can use `mdBook` to build and serve the documentation.

```sh
# Install mdBook
cargo install mdbook

# Build the documentation
mdbook build

# Serve the documentation locally
mdbook serve
```
